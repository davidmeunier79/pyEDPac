# pyEDPac

## installation

See or use requirements.txt


## All the required files for running a simulation are in examples/

### Full runs:

- run_population.py : the full simulation
- run_population_novisu.py : same, without graphical components
- run_population_novisu_joblib.py : no graphics, using parralel computations

### Testing

- test_evonetwork.py : test network function
- test_ga_training.py: test genetic algorithm
- test_population.py : test population creation
- test_PySide.py : test if pyside is working as expected
- test_simple_network.py : test basic neuron/synapse
- test_zoo_and_behaviour.py : test zoo and input
- test_zoo.py : test zoo

### Testing visu

- test_visu_pixel.py : test basic visualisation
- test_visu_zoo_input.py : test zoo and input (visual components)
- test_visu_zoo_input_network.py : test all visual components
